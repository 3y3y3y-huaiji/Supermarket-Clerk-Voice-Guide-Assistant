# 超市店员语音引导助手

一个基于语音识别和自然语言处理的超市店员辅助工具，帮助店员快速响应顾客需求。

## 功能特性
- 语音识别顾客需求
- 智能推荐商品位置
- 库存查询
- 促销信息查询
- 顾客引导

## 技术栈
- Python
- 语音识别API
- 自然语言处理
- 本地数据库

## 安装
```bash
git clone https://github.com/3y3y3y-huaiji/Supermarket-Clerk-Voice-Guide-Assistant.git
cd Supermarket-Clerk-Voice-Guide-Assistant
pip install -r requirements.txt
```

## 使用
```bash
python main.py
```
